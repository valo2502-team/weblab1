# Team-Shop - онлайн-магазин техніки

## Короткий опис
Team-Shop - невеликий онлайн-магазин техніки. Клієнти можуть переглядати товари через веб-браузер, а бекенд на Django забезпечує доступ до бази даних (SQLite). Система відображає список товарів та дозволяє адміністратору додавати або редагувати товари.  

Система спроектована так, щоб у майбутньому можна було додати авторизацію, платежі та повідомлення.

## Користувачі (ролі)
- **Клієнт**: переглядає товари
- **Адміністратор**: додає та редагує товари
- **Майбутні ролі**: авторизовані користувачі з доступом до додаткових функцій (Auth Service)

## Основні сценарії використання
1. Клієнт натискає "Load items" і бачить список товарів.
2. Адміністратор додає новий товар через бекенд.
3. Клієнт фільтрує товари за категоріями.
4. Адміністратор редагує існуючий товар.
5. Клієнт отримує повідомлення про оновлення каталогу (майбутній модуль).

## Функціональні вимоги
- GET /items API повертає список товарів.
- Додавання та редагування товарів.
- Відображення товарів на HTML сторінці.
- **Планується**: реєстрація та логін користувачів (Auth Service), повідомлення (Notifications).

## Нефункціональні вимоги
- Відповідь API < 200ms.
- Сумісність з сучасними браузерами.
- Простий і зрозумілий інтерфейс.

## Обмеження
- Система не реалізовує оплату.
- Реєстрація користувачів поки не реалізована (планується у майбутньому).
