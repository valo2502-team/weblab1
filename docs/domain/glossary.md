# Glossary (Ubiquitous Language)

### User
Особа, яка користується системою і має роль.

### Task
Одиниця роботи з описом, статусом, пріоритетом і виконавцем.

### Status
Стан завдання: todo → in_progress → done.

### Priority
Важливість завдання: low / medium / high.

### Comment
Повідомлення, яке користувач залишає під завданням.

### Assignment
Процес, коли завдання прив'язується до виконавця.
